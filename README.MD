In order to make this project run, you will need to follow the instructions mentioned below:
- Run in a shell (with the path the project directory) ```pip install -r requirements.txt```
- Run jupyter notebook in your local machine (if you want to) with the following command ```jupyter notebook --notebook-dir=DIR_NAME```
- Open the Jupiyer Notebook file and run each cell by order or run all by clicking in *Cell* and *Run all*.
