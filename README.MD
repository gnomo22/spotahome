## Prerequisites
- It is recommended to use Python 3.8
- You should have installed Jupyter Notebook
- If you do not want to run the code in your machine, you could directly explore the Notebook in git. However, bear in mind that this will not show any of the plots.

## How to run it
- Open a terminal (with the path the project directory).
- Run ```pip install -r requirements.txt```
- Run Jupyter Notebook with the following command ```jupyter notebook --notebook-dir=DIR_NAME```
- Open the Jupiter Notebook file and run each cell by order or run all by clicking in *Cell* and *Run all*.
